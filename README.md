# Sunbed

Angular app to sketch sunbeds.

![Screenshot of a sketched sunbed](https://raw.githubusercontent.com/Julien-Marcou/sunbed/main/src/assets/sunbed.png)

## Requirements

- Node.js `v18`
- Angular CLI `v14`

## Dev

- `ng serve`


## Lint

- `ng lint`
- `ng lint --fix`

## Build

- `ng build`
